# MarketPro Guide & Screener

A comprehensive guide to Indian Stock Market screening, featuring a **Real-time Screener** built with Python & Streamlit.

## Features
- **Educational Guide**: Learn how to find Multibaggers and Swing Trades using Screener.in.
- **Live Screener**: Fetch real-time data for top Indian stocks using Yahoo Finance API.
- **Interactive Charts**: Visualize growth trajectories and volume patterns.

## How to Run Locally

1.  **Install Python**: Make sure you have Python installed.
2.  **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```
3.  **Run the App**:
    ```bash
    streamlit run app.py
    ```

## How to Deploy (Free)

The best way to host this for free is **Streamlit Community Cloud**.

1.  **Push to GitHub**:
    -   This project is already initialized as a git repo.
    -   Push it to your GitHub account: `git push -u origin main`.
2.  **Connect Streamlit**:
    -   Go to [share.streamlit.io](https://share.streamlit.io/).
    -   Sign in with GitHub.
    -   Click "New App".
    -   Select your repository (`my-Screener`), branch (`main`), and file (`app.py`).
    -   Click **Deploy**.

Your app will be live in minutes!
