# MarketPro Guide & Screener

A comprehensive guide to Indian Stock Market screening, featuring a **Real-time Screener** built with Python & Streamlit.

## Features
- **Educational Guide**: Learn how to find Multibaggers and Swing Trades using Screener.in.
- **Live Screener**: Fetch real-time data for top Indian stocks using Yahoo Finance API.
- **Interactive Charts**: Visualize growth trajectories and volume patterns.

## How to Run Locally

1.  **Install Python**: Make sure you have Python installed.
2.  **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```
3.  **Run the App**:
    ```bash
    streamlit run app.py
    ```
